Downloading and installing the free sample solutions for Chapters 2 & 3

  1. You can check the current revision number here.

  2. Before installing the solutions guide, you must have the JDK (1.2 or 1.3) downloaded and properly installed. You can download the JDK from java.sun.com. Make sure you have '.' in your CLASSPATH.

  3. Click here to download TIJ2-Sol-sample.jar, which contains the sample solutions. Note that you may need to right-click and select "Save target as."

  4. Under Windows, double click the JAR file to begin the installation process. On other platforms, you must explicitly run the program by saying:
    java -jar TIJ2-Sol-sample.jar

  5. A program will run and a dialog box will appear. You must type the password into this dialog box. For the free trial version, the password is FreeSample. Then press the button that says "Write TIJ2-Sol-sample.zip". This will write the file whether or not you've put in the correct password.

  6. You can now unzip TIJ2-Sol-sample.zip using the Zip tool of your choice. If you get an error when trying to unzip the file, it means you did not use the correct password, so you should go back and run the Jar file again using the password FreeSample.

  7. Once you've unzipped the file, you'll have a code directory (with makefiles and a master makefile) and an html directory, containing the pages with code and annotations. You can open the html book by viewing the index.html file with your web browser.

  8. To compile the code using make, you must first download and install Gnu make from Simtel for DOS/Windows. You can find general information about make and places to download it for other machines at the Gnu make page. Note that Linux automatically installs Gnu make.

  9. Move to the code directory. To a command prompt, type
    make dos
    if you are running DOS/Windows or
    make linux
    if you are running Linux or Unix.

  10. All the code should compile without make aborting. In addition, all the main()s will be run to verify that the programs behave properly.
