Trail: JDK(TM) 1.1 -- And Beyond!

Lesson: A Preview of Things to Come

The Java engineering team plans to add many features to the JDK in an upcoming release. As of this writing, the release name is 1.2. As you read this preview, please keep in mind that it's always possible that a feature we tell you about will be changed or moved to a different release. Such is the nature of software!

The following sections briefly describe the features that are planned for the JDK 1.2 release. In addition to the features in this list, the Java engineering team plans to add a standard extensions architecture and enhance JAR, JNI, and reflection.

  • Security Enhancements
  • Java Foundation Classes (JFC)
  • Collections
  • Enhancements to the JavaBeans Architecture
  • Input Method Framework
  • Package Version Identification
  • RMI Enhancements
  • Serialization Enhancements
  • Weak References
  • Audio Enhancements
  • Java IDL
  • Performance Enhancements
  • Threads Changes
