Lesson: A Preview of Things to Come

JavaBeans Enhancements

Glasgow is the code-name for the next release of the JavaBeans component model specification. Glasgow provides developers with standard means to create more sophisticated Java components and applications that offer their customers more seamless integration with the rest of their runtime environment, such as the desktop of the underlying OS or the browser. To achieve this, Glasgow adds three new capabilities to the JavaBeans component model:
  • An extensible runtime containment and services protocol
  • a drag and drop subsystem for JFC
  • the JavaBeans activation framework.
