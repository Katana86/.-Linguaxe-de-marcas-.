Trail: Drag and Drop

One day we'll have a trail here. For now, here are some links to resources and examples:

JavaBeans home page
Indirectly links to the Drag and Drop specification; might have other information related to Drag and Drop.

SwingDrop.java
A rough example showing plain-text drag-and-drop with a JTextArea. Prints diagnostics.

A Drag and Drop Example by Sheetal Gupta
This example contains two lists. You can drag an item from either list and drop it onto the other list. Demonstrates that a single component can be both a Drag Source and a Drop Target. Prints diagnostics. You can also search the Web sites listed in Online Resources for Java Programmers for drag-and-drop articles and examples. If you can recommend any other examples or resources, please tell us.
