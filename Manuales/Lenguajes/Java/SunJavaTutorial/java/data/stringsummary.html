Other Interesting Features

String and StringBuffer provide several other useful ways to manipul ate string data, including concatenation, comparison, substitution, and conversion to upper and lower case. java.lang.String and java.lang.StringBuffer summarize and list all of the methods and variables supported by these two classes.
