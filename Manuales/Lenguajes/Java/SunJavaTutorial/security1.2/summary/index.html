Trail: Security in Java 2 SDK 1.2

Lesson: Summaries

This section is a collection of summaries:

Glossary provides definitions and usage descriptions of important general security terms (such as "digital signature," "public key," and "private key") and of security terms specific to the Java platform (such as "permission" and "policy").

API Summary discusses what the JDK Security API provides, and describes the main API classes and interfaces.

Tools Summary provides brief overviews of each of the security tools, and lists their most common commands and options.

Security-related Files Summary provides an overview of the security-related files that are built into JDK 1.2 as well as files that you may optionally create.
