Applet "Paseo aleatorio"

En este applet la "alteración" vaga de forma aleatoria por una rejilla de cuadrados coloreados. Inicialmente la rejilla es negra. Cada vez que la alteración visita un cuadrado, este se convierte brillando ligeramente con un toque de verde, hasta determinados brillos máximos. Eventualmente, se supone, el cuadrado entero puede llegar a alcanzar un uniforme color verde.

Para realizar este "paseo aleatorio", la alteración va cambiando de dirección -- arriba, abajo, derecha e izquierda-- de forma aleatoria. Esto se repite de forma indefinida. ( Tenga en cuenta que el borde superior se considera unido al inferior, y el derecho al izquierdo, de tal forma que cuando el cuadrado sale por un extremo, aparece por el opuesto).
