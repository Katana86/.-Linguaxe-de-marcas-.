Applet "El cuadrado saltarín"

El applet de esta pagina ilustra dos ideas: Responde a los eventos producidos por el mouse cuando el usuario hace clic  en cualquier sitio del applet. Además hay una segunda hebra que esta funcionando continuamente para presentar el tiempo transcurrido y hacer que el cuadrado salte aunque el usuario no genere ningún evento.

Pruebe de hacer clic sobre el cuadrado.Cuando acierte, sumara puntos a "Hit".Si  no acierta, y hace el clic donde no esta el cuadrado, sumará al contador de "miss."
