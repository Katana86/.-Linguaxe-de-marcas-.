Capítulo 8

Entrada/salida avanzada y Excepciones

DESGRACIADAMENTE ES MUY CIERTO QUE cuando los programas estan funcionando, aparecen errores. Existen errores de programación. como un indice de serie fuera de rango, o un intento de dividir un numero por cero. Pero tambien hay errores sobre los que el programador tiene menos control, o incluso ningun control. como los que ocurren cuando el usuario escribe una palabra donde el programa espera un numero, o cuando el sistema provoca un error de memoria.

A menudo. cuando ocurre un error, el programa simplemente aborta (o incluso peor, produce resultados incorrectos). Sin embargo Java proporciona unos mecanismos para controlar estos errores y otras "condiciones extrañas". Esta capacidad de manejar errores es uno de los temas de este capítulo. El otro tema son las facilidades de entrada/salida integradas en Java, que estan implementads en el objeto llamado Streams. Pero es imposible utilizar este objeto sin entender un poco las excepciones.
