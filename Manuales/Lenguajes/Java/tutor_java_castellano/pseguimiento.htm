Seguimiento de ejercicios:

Con este contrato, además de los servicios de tutoría, podrá, y deberá, enviar los ejercicios indicador en los formularios de practicas, su tutor revisara su trabajo, le corregirá los errores y le señalara los temas que le pueden convenir repasar. Condiciones especiales de este servicio:

Consideramos que la persona que se acoge a la corrección de exámenes esta apostando seriamente por nuestro servicio y por ello, pensamos que es conveniente establecer una normativa mas rígida.

  • Los trabajos se presentaran de forma ordenada, no se realizara la corrección de un trabajo, hasta que el anterior se haya pasado correctamente, o se hayan utilizado los tres intentos.
  • Una vez se pase de nivel, no se corregirán nuevamente los ejercicios anteriores, si bien el usuario puede realizar cuantas consultas considere oportunas.

Dado que esto se hace únicamente en interés del alumno, bastara con una nota suya para que esta normativa quede sin efecto.
