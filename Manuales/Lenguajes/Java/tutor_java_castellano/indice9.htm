Capitulo 9

Desde Java hasta C++

JAVA ES EL LENGUAJE DE PROGRAMACION MAS MODERNO. Aunque haya conseguido ya el ser reconocido como un importante lenguaje de programacion, no deja de ser cierto que la programacion mas "seria", se esta realizando en C y C++. Afortunadamente, estos lenguajes comparten muchas caracteristicas con Java.Al antigui C, le faltan muchas caracteristicas de orientacion a objetos. C++ es un lenguaje mucho mas completo que extiende C con clases, objetos y otras posibilidades. Este capitulo, es una breve introduccion al C++ para alguien que conoce Java. La cobertura es incompleta y solo es el punto de partida para empezar a aprender C++.

Encontrara lo basico (Diseño de programacion) es practicamente identico en Jaca y en C++. Sin embargo las filosogias de programacion y laa estructura de programas grandes ("Programacion en detalle"), en C++. es bastante diferente a Java.
