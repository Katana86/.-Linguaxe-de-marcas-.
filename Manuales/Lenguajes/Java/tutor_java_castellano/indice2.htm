Capitulo 2

Programacion :
Variables. I/O, y estructuras de control

A BAJO NIVEL ( a nivel de código maquina), el ordenador realiza unicamente sencillas operaciones. Para que el ordenador realice tareas complejas, se han de montar juntas gran numero de estas operaciones. Esta tarea  debe ser escrita completa y perfectamente detallada por el programador. Crear programas complejos, realmente no es facil, pero la dificultad, puede ser controlada parcialmente programando un conjunto de sencillas estructuras. El diseño en conjunto de estas estructuras, es lo que llamo "Diseño de programa".

Programacion detallada, que tambien es llamada a veces, codificacion, viene a referirse a completar detalladamente los diseños. Los detalles, han de ser explicitos, instrucciones paso a paso para realizar tareas relativamente pequeñas. Cuando codifica, esta trabajando relativamente "cerca de la maquina", con muchos de los mismos conceptos que se deben usar en el lenguaje maquina: posiciones de memoria, operaciones aritmeticas, bucles y decisiones. En un lenguaje de alto nivel como Java, podrá trabajar con estos conceptos,  a un nivel a bastante distancia del lenguaje maquina. No obstante, debera preocuparse acercaa de definir todos los detalles con total exactitud.

Este capítulo examina las facilidades que da el lenguaje Java para la programacion.
