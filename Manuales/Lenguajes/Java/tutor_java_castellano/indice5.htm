Capitulo 5

Applets, HTML y el web

Java es un lenguaje de programación diseñado para el mundo actual co los ordenadores unidos por una red. Los programas en Java, pueden descargarse por la red para que corran en algun ordenador que implemente la maquina virtual Java. Este lenguaje con su formato de applet que se puede fusionar en una pagina Web, puede llegar a convertirse en parte integral de la World Wide Web, el servicio de Internet mas interesante y que mas rapidamente crece.

Este capítulo presenta las bases de la programacion de applets de Java. Tambien explica que es el Web y los fundamentos del HyperText Markup Languaje (HTML), el lenguaje que se emplea para escribir pagina Web. Las explicaciones sobre applets continuaran en los siguientes capítulos.
