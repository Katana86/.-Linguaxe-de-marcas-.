Computer Science 124:

Curso de introducción a la programación en JAVA

 

Bienvenidos al curso On-Line de Introducción a la programación. Este curso, fue desarrollado durante el invierno de 1.998 en Hobart and William Smith Colleges por (David Eck, email: eck@hws.edu). El curso, es una introducción a la teoría y las técnicas de programación , usando Java como lenguaje, sin necesitar conocimientos previos. Tenga en cuenta que es la programación, mas que Java en si mismo que es el   objeto del módulo CPSC122. Si quiere conocer mas acerca de este curso, vea las paginas de información. (También puede leer el libro The Most Complex Machine escrito para servir de introducción a la ciencia de la computación.)

El autor da libertad de uso de texto, siempre que no se reclamen créditos, o se venda sin su permiso. Para la traducción al castellano, quiero hacer constar las mismas condiciones.

Conjuntamente con el texto, puede ir siguiendo los ejercicios que se usaron durante el curso.Están disponibles para bajarlos por separado.

Nota: Este texto analiza solo la versión 1.0 de Java y puede no aclarar algún tema de las siguientes versiones.

(Este texto ha sido ligeramente modificado de la versión original que fue usada en 1996. Esta versión se puede encontrar en ingles aquí.)

Notas del traductor:

        Durante la traducción, y para facilitar la lectura, intentare dejar en ingles todos aquellos términos que por su constante uso, son mas comprensibles en ese idioma. Así mismo, procurare ir dejando notas a pie de pagina acerca de su significado, que no de su traducción, ya que no tendría demasiado sentido.

        Los capítulos, se irán publicando, a medida que se vayan traduciendo. Para mantener la coherencia del curso, se publicara a la vez el capitulo, el trabajo de laboratorio que corresponda, y todos los temas relacionados.

        La traducción de un curso sobre informática, si se quiere hacer bien, es muy compleja, por lo que desde este punto, invito a todos los que sigan estas líneas, a corregirme todas aquellas expresiones que parezcan inadecuadas. Quiero hacer hincapié que por mi nacionalidad (española), estoy utilizando mi castellano que se que no siempre es totalmente comprensible para otros hispanohablantes, sin embargo, pienso que entre todos, podemos hacer un esfuerzo para que el trabajo sea lo mas claro posible, y cuando no, realizar capítulos distintos para cada nacionalidad.
