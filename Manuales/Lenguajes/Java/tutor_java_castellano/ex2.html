Applet "Prueba de transformación"

El applet que se presenta en esta pagina es una rejilla de cuadrados coloreados. Inicialmente se generan cuadrados de colores distintos. A continuación se selecciona de forma aleatoria un cuadrado, después se selecciona aleatoriamente uno de sus cuatro vecinos, y el color del cuadrado seleccionado se convierte al color se su vecino. Esto se repite de forma infinita. (Observe que para seleccionar vecinos, el borde superior aparece unido al inferior y el borde derecho al  izquierdo.)
