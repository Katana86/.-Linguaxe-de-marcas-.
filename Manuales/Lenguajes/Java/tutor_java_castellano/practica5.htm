Algunas estadísticas sencillas

Las estadísticas son números que resumen o abstraen algunas propiedades de un conjunto de números (potencialmente grandes). Por ejemplo, es una estadística la que representa la longitud media de un conjunto de números. La desviación estándar es una estadística que define de alguna manera cuan ampliamente dispersos están los números analizados. Dado que estas estadísticas, a menudo necesitan ser calculadas por programas, es lógico el explicar algo sobre como calcularlas por clases y poderlas usar una y otra vez.

La carpeta "StatCalc Starter" contiene un programa principal que calcula algunas estadísticas para un conjunto de números entrados por el usuario. (Lo puede leer aquí). Los cálculos los realiza "a mano". Debe leerse este programa para entender lo que hace y como lo hace (No necesita entender como trabaja la formula de la desviación estándar). Un detalle interesante es que todas las estadísticas se calculan sin mantener ningún control sobre todos los números entrados. Cada vez que se le un numero, se actualizan algunas variables, el programa solo necesita arrastrar la suma de los números entrados, y no los números en si.

El el proyecto "StatCalc Starter", también encontrará el fichero StatCalc.java, que define la clase llamada StatCalc ( lo puede ver aquí). Su trabajo, es reescribir el programa para utilizar objetos pertenecientes a la clase StatCalc toda ella dedicada a cálculos estadísticos. Añadir la variable stats de tipo StatCalc al programa. Leer la definición de la clase StatCalc para ver que métodos ofrece. Deberá llamar stats.enter(), stats.getCount(), stats.getStandardDeviation(), stats.getMaximum(), y stats.getMinimum().  Emplee estos métodos para entrar los datos en el conjunto y para calcular las estadísticas del conjunto.

Deberá borrar la mayoría de las variables y casi todo el código original del programa. La versión que escriba deberá ser bastante mas pequeña que la original.
